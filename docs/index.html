Twitter Sentiment Analyzer

This analyzer will allow a user to input a given twitter handle. The twitter page will be scraped for tweets, which will be classified into different sentiments by a pretrained sentiment analyzer, and the sentiments will be aggregated to produce an overall sentiment score for the account.

Expected features:

  • An about section explaining how the analyzer model works
  • An input field for a twitter handle
  • A scraping mechanism to collect tweets posted from the handle
  • A pretrained sentiment analysis model to classify the tweets as positive, negative, etc (may change or vary)
